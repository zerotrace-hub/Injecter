# INJECTER - Android Payload Injection Framework

A professional Android payload injection tool using APKTool and Metasploit Framework.

## Features
-  Create Android meterpreter payloads
-  Inject payloads into existing APKs using APKTool
-  Auto-sign APKs with custom keystore
-    Latest APKTool version support
-    Clean and professional interface

## Installation

```bash
# Clone repository
git clone https://github.com/yourusername/Injecter.git
cd Injecter

# Make scripts executable
chmod +x setup.sh Injecter.sh exploit.sh

# Install dependencies
sudo ./setup.sh

# Run the tool
./Injecter.sh
